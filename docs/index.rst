splinenorm
==========

Lightweight tools for **B-spline continuum normalization** and weighted linear
fitting of astronomical spectra.

Installation
------------

.. code-block:: bash

   pip install splinenorm

For development:

.. code-block:: bash

   git clone https://github.com/<your-org>/splinenorm.git
   cd splinenorm
   pip install -e ".[dev]"

Examples
--------

Step-by-step Jupyter notebooks and diagnostic figures live in the
:doc:`examples` page. Bundled example spectra are in ``docs/data/``.
